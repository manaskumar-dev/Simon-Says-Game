# Simon Game 🎮

A simple Simon Says game built using HTML, CSS, and JavaScript.

## How it works
- Press any key to start the game
- Watch the sequence of flashes
- Repeat the sequence by clicking the buttons
- The sequence increases with each level
- Game ends on wrong input

## Features
- Random sequence generation
- Button flash effects
- User input validation
- Game over screen effect

## Tech Used
- HTML
- CSS
- JavaScript

## About
This project was built from scratch to practice JavaScript fundamentals like DOM manipulation, event handling, and game logic.
